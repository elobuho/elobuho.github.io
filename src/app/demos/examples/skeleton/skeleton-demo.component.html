Skeleton directive

A structural directive with multiple inputs to adjust the placeholder component. Using skeletons prevents layout shifts caused by asynchronous content loading, which in turn improves user experience and boosts SEO scores.

The skeleton directive requires a boolean value to control the placeholder. Set it to false once the data is loaded to display the real content.

The height in rem units is supported, , as well as the repeat option for multiplying placeholders and the class input to match the styling of the real block.

Here is the real-world example:

*uiSkeleton="isLoading(); class: 'size-64'"

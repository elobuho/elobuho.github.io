Carousel component

A classic slider built with modern techniques. Features include image lazy-loading, scroll-snapping, and full responsiveness. It is also server side rendering compatible.

By default carousel component has the following settings:

[visibleItems]="1"
[buttons]="true"
[behaviour]="'smooth'"
[pager]="true"
[duration]="7000"

Let's show two items at once. CSS variables can be used to apply a negative margin (to counteract the spacing) and tweak the pagination. 

--carousel-scroll-area-margin: -16px;
--carousel-pager-active-color: #f00;

Any element can be marked with uiButtonPrev and uiButtonNext directives to point out custom buttons.
Tip: Non visible on mobiles.
